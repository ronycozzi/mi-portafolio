Introducción a JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Es uno de los tres pilares del desarrollo web, junto con HTML y CSS, y se ejecuta en el navegador.

Sintaxis Básica de JavaScript

JavaScript utiliza una sintaxis similar a otros lenguajes de programación. Aquí tienes un ejemplo básico:


// Esto es un comentario en JavaScript
let nombre = "Juan"; // Declaración de una variable
console.log(nombre); // Imprime el valor de la variable en la consola

Variables y Tipos de Datos

JavaScript permite declarar variables usando var, let, o const. Los tipos de datos incluyen:

  • String: Texto, por ejemplo "Hola Mundo".
  • Number: Números, por ejemplo 42 o 3.14.
  • Boolean: Valores lógicos, true o false.
  • Array: Lista de valores, por ejemplo [1, 2, 3].
  • Object: Estructura que permite agrupar valores bajo propiedades, por ejemplo {nombre: "Juan", edad: 30}.
  • Undefined: Representa una variable que ha sido declarada pero no se le ha asignado un valor.
  • Null: Representa la ausencia intencional de cualquier valor.

let edad = 25; // Número
let esMayorDeEdad = true; // Booleano
let nombres = ["Ana", "Pedro", "Luis"]; // Array
let persona = {nombre: "María", edad: 22}; // Objeto
let direccion; // Undefined
let cuenta = null; // Null

Operadores en JavaScript

JavaScript incluye varios operadores para realizar operaciones matemáticas y lógicas:

  • Aritméticos: +, -, *, /, % (suma, resta, multiplicación, división, módulo)
  • De comparación: ==, ===, !=, !==, >, < (igualdad, identidad, desigualdad, mayor que, menor que)
  • Lógicos: && (AND), || (OR), ! (NOT)
  • Asignación: =, +=, -=, *=, /= (asignación y asignación combinada)

let x = 10;
let y = 5;
console.log(x + y); // 15
console.log(x > y); // true
console.log(x && y < 15); // true
x += 5; // x ahora es 15

Estructuras de Control

Las estructuras de control permiten dirigir el flujo de ejecución de tu código. Las más comunes son:

  • if/else: Ejecuta un bloque de código si una condición es verdadera.
  • switch: Selecciona entre múltiples opciones basadas en el valor de una expresión.
  • for: Repite un bloque de código un número específico de veces.
  • while: Repite un bloque de código mientras una condición sea verdadera.
  • for...in: Itera sobre las propiedades enumerables de un objeto.
  • for...of: Itera sobre los valores de un objeto iterable, como un array.

let numero = 10;
if (numero > 5) {
    console.log("El número es mayor que 5");
} else {
    console.log("El número es 5 o menor");
}

for (let i = 0; i < 5; i++) {
    console.log(i); // Imprime números del 0 al 4
}

let persona = {nombre: "Ana", edad: 30};
for (let propiedad in persona) {
    console.log(propiedad); // Imprime "nombre" y "edad"
}

let arreglo = [10, 20, 30];
for (let valor of arreglo) {
    console.log(valor); // Imprime 10, 20, 30
}

Funciones en JavaScript

Las funciones permiten encapsular bloques de código para ser reutilizados. Puedes declararlas usando function o como funciones flecha:

  • Funciones declarativas: Se declaran utilizando la palabra clave function.
  • Funciones anónimas: Funciones sin nombre que se asignan a una variable.
  • Funciones flecha: Sintaxis más concisa para funciones, especialmente útiles en funciones anónimas.
  • Parámetros y argumentos: Las funciones pueden recibir datos (parámetros) y devolver un valor.

function saludar(nombre) {
    return "Hola " + nombre;
}
console.log(saludar("Juan")); // "Hola Juan"

const sumar = (a, b) => {
    return a + b;
}
console.log(sumar(5, 3)); // 8

DOM (Document Object Model)

El DOM es una representación estructural del documento HTML que permite a JavaScript interactuar con los elementos de la página:

  • document.getElementById(): Selecciona un elemento por su ID.
  • document.querySelector(): Selecciona el primer elemento que coincida con un selector CSS.
  • element.textContent: Cambia el texto de un elemento.
  • element.innerHTML: Cambia el HTML interno de un elemento.
  • element.style: Cambia el estilo CSS de un elemento.
  • element.classList: Manipula las clases CSS de un elemento.

let titulo = document.getElementById("titulo");
titulo.textContent = "Nuevo Título";
titulo.style.color = "blue";
titulo.classList.add("nuevo-estilo");

Manejo de Eventos

JavaScript puede responder a eventos como clics, movimientos del ratón, o envíos de formularios:

  • addEventListener(): Vincula un evento a un elemento.
  • event.preventDefault(): Previene el comportamiento predeterminado del evento.
  • event.stopPropagation(): Evita que un evento se propague a otros elementos.
  • Eventos comunes: click, submit, mouseover, keyup.

let boton = document.getElementById("miBoton");
boton.addEventListener("click", function(event) {
    alert("¡Botón clickeado!");
    event.preventDefault(); // Previene el comportamiento predeterminado
});

Ámbito de Variables y Closures

El ámbito determina la accesibilidad de las variables en diferentes partes del código. Existen tres tipos principales:

  • Ámbito global: Las variables declaradas fuera de cualquier función tienen ámbito global.
  • Ámbito de función: Las variables declaradas dentro de una función solo son accesibles dentro de esa función.
  • Ámbito de bloque: Las variables declaradas con let o const dentro de un bloque ({...}) solo son accesibles dentro de ese bloque.

Los closures son funciones que recuerdan el entorno léxico en el que fueron creadas, lo que permite que una función interna acceda a las variables de la función externa incluso después de que esta haya terminado de ejecutarse.


function crearContador() {
    let contador = 0;
    return function() {
        contador += 1;
        return contador;
    }
}

const contador1 = crearContador();
console.log(contador1()); // 1
console.log(contador1()); // 2
const contador2 = crearContador();
console.log(contador2()); // 1

Promesas y Async/Await

Las promesas y async/await son herramientas en JavaScript para manejar operaciones asíncronas de manera más eficiente:

  • Promesas: Representan un valor que puede estar disponible ahora, en el futuro, o nunca.
  • Async/Await: Permiten escribir código asíncrono de manera más clara y secuencial.

// Promesas
let promesa = new Promise((resolve, reject) => {
    let exito = true;
    if (exito) {
        resolve("La operación fue exitosa");
    } else {
        reject("La operación falló");
    }
});
promesa.then(resultado => console.log(resultado))
       .catch(error => console.log(error));

// Async/Await
async function obtenerDatos() {
    try {
        let respuesta = await fetch("https://api.ejemplo.com/datos");
        let datos = await respuesta.json();
        console.log(datos);
    } catch (error) {
        console.log("Error:", error);
    }
}
obtenerDatos();
