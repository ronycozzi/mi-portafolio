Introducción a HTML

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear y estructurar documentos web. Es fundamental para todo desarrollador web, ya que permite definir la estructura y el contenido de las páginas web.

Estructura Básica de una Página HTML

Una página HTML está compuesta por varias etiquetas que definen su estructura. A continuación, se muestra un ejemplo de la estructura básica de un documento HTML:

                
                    <!DOCTYPE html> 
                    <html lang="es"> 
                        <head> 
                            <meta charset="UTF-8"> 
                            <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                            <title>Título de la página</title> 
                            <link rel="stylesheet" href="styles.css"> 
                        </head>
                        <body> 
                            <header> 
                                <h1>Encabezado Principal</h1>
                            </header>
                            <nav> 
                                <ul> 
                                    <li><a href="#">Inicio</a></li>
                                    <li><a href="#">Sobre nosotros</a></li>
                                    <li><a href="#">Contacto</a></li>
                                </ul>
                            </nav>
                            <main> 
                                <section> 
                                    <h2>Subtítulo</h2>
                                    <p>Este es un párrafo de texto.</p>
                                </section>
                            </main>
                            <footer> 
                                <p>Derechos reservados © 2024</p>
                            </footer>
                        </body>
                    </html>

Etiquetas HTML Comunes y sus Usos

Estas son algunas de las etiquetas HTML más comunes que encontrarás al desarrollar una página web:

  • <h1> a <h6> - Define encabezados de diferentes niveles, desde el más importante (<h1>) hasta el menos importante (<h6>).
  • <p> - Define un párrafo de texto.
  • <a> - Define un hiperenlace que puede llevar a otras páginas o secciones.
  • <img> - Inserta una imagen en la página.
  • <div> - Contenedor genérico que agrupa otros elementos.
  • <span> - Contenedor en línea para aplicar estilos a partes de texto o elementos.
  • <ul>, <ol>, <li> - Listas no ordenadas, ordenadas y elementos de lista.
  • <table>, <tr>, <td>, <th> - Etiquetas para crear tablas.
  • <form> - Contenedor para formularios que capturan datos del usuario.
  • <input> - Campo de entrada de datos en un formulario.
  • <textarea> - Área de texto para entradas más largas.
  • <button> - Botón que puede realizar acciones como enviar un formulario.
  • <select>, <option> - Lista desplegable con opciones seleccionables.
  • <label> - Etiqueta asociada a un elemento de formulario para mejorar la accesibilidad.
  • <header>, <footer>, <article>, <section>, <aside> - Etiquetas semánticas que definen la estructura de una página.
  • <iframe> - Permite insertar otro documento HTML dentro de la página actual.
  • <script> - Etiqueta para incluir código JavaScript.
  • <link> - Conecta recursos externos, como hojas de estilo CSS, con la página.
  • <meta> - Define metadatos sobre la página HTML, como la codificación de caracteres o la descripción de la página.

Ejemplos de Código con Etiquetas Comunes

A continuación se presentan ejemplos prácticos de cómo utilizar las etiquetas HTML mencionadas:

Ejemplo de un enlace (<a>)

<a href="https://www.ejemplo.com">Visita nuestro sitio</a>

Ejemplo de una imagen (<img>)

<img src="imagen.jpg" alt="Descripción de la imagen">

Ejemplo de una lista no ordenada (<ul> y <li>)

<ul>
<li>Elemento 1</li>
<li>Elemento 2</li>
<li>Elemento 3</li>
</ul>

Ejemplo de una tabla (<table>, <tr>, <td>, <th>)

<table>
<tr>
<th>Encabezado 1</th>
<th>Encabezado 2</th>
</tr>
<tr>
<td>Dato 1</td>
<td>Dato 2</td>
</tr>
</table>

Formularios en HTML

Los formularios son esenciales para interactuar con los usuarios y capturar datos. Un formulario básico incluye etiquetas como <form>, <input>, y <button>. Aquí tienes un ejemplo:

<form action="submit_form.php" method="post">
<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">
<label for="email">Email:</label>
<input type="email" id="email" name="email">
<button type="submit">Enviar</button>
</form>

Elementos Semánticos en HTML5

HTML5 introdujo varios elementos semánticos que ayudan a estructurar mejor el contenido de la página y a mejorar la accesibilidad:

  • <header> - Define un encabezado para un documento o sección.
  • <nav> - Define una sección de navegación.
  • <section> - Agrupa contenido relacionado en una sección de la página.
  • <article> - Contenido independiente que puede ser distribuido o reutilizado.
  • <aside> - Contenido que está relacionado con el contenido principal, pero que no es parte de él directamente.
  • <footer> - Define el pie de página para un documento o sección.
