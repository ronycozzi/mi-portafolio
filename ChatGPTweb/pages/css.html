Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño que controla la presentación visual de los elementos HTML en una página web. Con CSS, puedes definir colores, fuentes, espaciamientos, y disposiciones en pantalla.

Sintaxis de CSS

Una regla CSS está compuesta por un selector y un bloque de declaraciones:

                
                    selector {

                        propiedad: valor;

                    }

Por ejemplo:

                
                    p {

                        color: blue;

                        font-size: 16px;

                    }

Selectores CSS

Los selectores permiten seleccionar los elementos HTML a los que se aplicarán los estilos. Algunos selectores comunes incluyen:

  • Selector de etiquetas: Selecciona todos los elementos de un tipo específico (por ejemplo, p selecciona todos los párrafos).
  • Selector de clases: Selecciona todos los elementos con una clase específica (por ejemplo, .clase).
  • Selector de ID: Selecciona un elemento único con un ID específico (por ejemplo, #id).
  • Selectores de atributo: Seleccionan elementos basados en un atributo y su valor (por ejemplo, input[type="text"]).
  • Selectores de pseudoclase: Aplican estilos a un estado específico de un elemento, como :hover para cuando el mouse pasa por encima.

Propiedades CSS Comunes y Detalladas

Aquí se detalla cada propiedad CSS común, con sus subpropiedades y ejemplos de uso:

  • color: Establece el color del texto.
    • Ejemplo: color: red; (Cambia el color del texto a rojo)
  • background-color: Define el color de fondo de un elemento.
    • Ejemplo: background-color: lightblue; (Cambia el fondo a azul claro)
  • font-size: Establece el tamaño de la fuente.
    • Ejemplo: font-size: 20px; (Aumenta el tamaño del texto a 20 píxeles)
  • margin: Establece el margen alrededor de un elemento.
    • Subpropiedades:
      • margin-top: Margen superior
      • margin-right: Margen derecho
      • margin-bottom: Margen inferior
      • margin-left: Margen izquierdo
    • Ejemplo: margin: 10px; (Aplica un margen de 10 píxeles en todos los lados)
    • Ejemplo: margin: 10px 20px; (Aplica 10px arriba y abajo, 20px a la derecha e izquierda)
  • padding: Define el espacio entre el contenido de un elemento y su borde.
    • Subpropiedades:
      • padding-top: Padding superior
      • padding-right: Padding derecho
      • padding-bottom: Padding inferior
      • padding-left: Padding izquierdo
    • Ejemplo: padding: 15px; (Aplica un padding de 15 píxeles en todos los lados)
    • Ejemplo: padding: 10px 20px; (Aplica 10px arriba y abajo, 20px a la derecha e izquierda)
  • border: Define el borde alrededor de un elemento.
    • Subpropiedades:
      • border-width: Ancho del borde
      • border-style: Estilo del borde (solid, dashed, etc.)
      • border-color: Color del borde
    • Ejemplo: border: 1px solid black; (Aplica un borde de 1px, sólido y negro)
    • Ejemplo: border: 2px dashed red; (Aplica un borde de 2px, discontinuo y rojo)

Modelo de Caja (Box Model)

El modelo de caja de CSS describe el diseño y la disposición de los elementos en una página web. Cada elemento se representa como una caja que tiene:

  • Contenido: El área donde se muestra el contenido (texto, imágenes, etc.).
  • Padding: El espacio entre el contenido y el borde.
  • Border: El borde alrededor del padding.
  • Margin: El espacio entre el borde y otros elementos.

Ejemplo visual:

                
                    .caja {

                        width: 200px;

                        height: 100px;

                        padding: 10px;

                        border: 5px solid black;

                        margin: 20px;

                    }

Layout y Flexbox

Flexbox es un modelo de layout en CSS que permite diseñar de manera eficiente distribuciones de elementos. Ejemplo básico:

                
                    .contenedor {

                        display: flex;

                        justify-content: space-between;

                    }


                    .item {

                        width: 100px;

                        height: 100px;

                        background-color: lightblue;

                    }

Propiedades clave de Flexbox incluyen:

  • display: flex; - Activa Flexbox en el contenedor.
  • justify-content - Controla la alineación de los elementos hijos a lo largo del eje principal (horizontal).
  • align-items - Controla la alineación de los elementos hijos a lo largo del eje transversal (vertical).
  • flex-direction - Define la dirección de los elementos hijos (fila, columna).

Media Queries

Las media queries permiten aplicar estilos específicos según el tamaño de la pantalla o el dispositivo. Son fundamentales para diseños responsivos:

                
                    @media (max-width: 600px) {

                        body {

                            background-color: lightblue;

                        }

                    }

Ejemplos Prácticos

Aquí se aplican todos los conceptos aprendidos en ejemplos más complejos, como la creación de un layout responsivo completo.

                
                    /* Ejemplo de layout responsivo */
                    .contenedor {

                        display: flex;

                        flex-wrap: wrap;

                    }


                    .item {

                        flex: 1 1 200px;

                        margin: 10px;

                        background-color: lightblue;

                    }


                    @media (max-width: 600px) {

                        .item {

                            flex: 1 1 100%;

                        }

                    }
